行列式

行列式基本概念

  • 把n个不同的数排成一列，叫做这n个数的全排列。一般来说，1 2 3 4 … n是n个数排列的标准次序。
  • 当n个数的任意排列中两个数的先后次序与标准次序不同时，就说有一个逆序。
  • 一个排列中所有的逆序总数称为这个排列的逆序数。
  • 如果排列的逆序数是奇数，该排列称为奇排列；如果排列的逆序数是偶数，该排列称为偶排列。
  • 对换：在排列中对调两个数的位置，其余数不动，得到一个新的排列，叫做一个对换。
  • 一个排列中任意两个元素对换，排列改变奇偶性。
  • n阶行列式定义：数表中有n^2^个数，取自不同行不同列元素乘积的和或差，称为n阶行列式。

    行列式性质

  • 行列互换，行列式的值不变，即行列式等于行列式的转置。
  • 互换行列式某两行或某两列，行列式改变符号。
  • 行列式有两行完全相同，则行列式为0.
  • 行列式某一行都乘以k，等于k乘以此行列式。
  • 行列式中某两行（列）对应元素成比例，行列式为0.
  • 行列式某一行（列）元素都是两数和，则可以拆成两个行列式的和。（n阶行列式，每个元素都是两个数的和，可以拆成2^n^个行列式的和）
  • 把行列式某一行（列）的k倍加到某一行（列）上，行列式不变。

    特殊行列式

  • 上三角行列式：主对角线以下元素全为0（不包括主对角线），行列式的值等于主对角线元素乘积。
  • 下三角行列式：主对角线以上元素全为0（不包括主对角线），行列式的值等于主对角线元素乘积。
  • 反上三角行列式：副对角线以上元素全为0（不包括对角线），行列式的值的绝对值等于主对角线元素乘积（n（n-1）/2决定符号）。
  • 反下三角行列式：副对角线以下元素全为0（不包括对角线），行列式的值的绝对值等于主对角线元素乘积（n（n-1）/2决定符号）。
  • 范德蒙行列式：第一行全为1（理解为x1，x2，x3….xm的零次方），第二行为第一行的一次方，后面以此类推，直到n-1次方（n*m）,该行列式的值采用数学归纳法求解得出。

    行列式的展开

  • 在n阶行列式中，把aij所在第i行第j列划去后，余下的n-1阶行列式，叫做aij的余子式，记作Mij。Aij=（-1）^i+j^Mij叫做aij的代数余子式。
  • 行列式的值等于它的任意一行（列）各元素与其对应代数余子式乘积的和。

    矩阵及其运算

    基本概念

  • 定义：由m×n个数aij排成的m行n列的数表，称为m行n列矩阵，简称m×n矩阵。
  • m×n称为矩阵A的型。
  • 行数与列数都等于n的矩阵称为n阶矩阵，或n阶方阵。
  • 只有一行元素的矩阵称为行矩阵（行向量），只有一列元素的矩阵称为列矩阵（列向量）。
  • 同型矩阵：两个矩阵的行数和列数分别对应相等。
  • 0矩阵：元素全为0的矩阵。
  • n阶对角矩阵：对角线以外元素全为0的方阵。特别的，当对角元素等于同一个数时，称为数量矩阵（纯量矩阵）；当对角元素等于1时，称为单位矩阵。
  • 非齐次线性方程组未知数系数构成的矩阵称为系数矩阵；常数项构成的矩阵称为常数项矩阵。两个矩阵放在一起，称为非齐次线性方程组的增广矩阵。

    矩阵的运算

  • 设A，B是两个同型矩阵，则两个矩阵相加（减）等于对应位置各元素相加（减）。
  • 矩阵的加减法满足交换律和结合律。
  • 矩阵的数乘：满足交换律，结果等于矩阵中每一个元素乘以k。
  • 矩阵相乘需要满足：第一个矩阵的列等于第二个矩阵的行，矩阵相乘不满足交换律。
  • 对于两个矩阵：AB=AC不能推出B等于C，AB=0不能推出A=0或B=0.
  • 矩阵乘法满足：结合律、分配律。
  • 数量阵与任何矩阵的乘法都是可以交换的（单位矩阵E可以与任意相乘矩阵交换）。
  • 矩阵的幂运算：必须保证矩阵为方阵，n个A相乘。
  • 通常，矩阵乘以一个向量可以解释为向量的旋转变换（线性变换）。
  • A^k^=0，称A为幂0阵。

    特殊矩阵

  • 对称矩阵：对于n阶方阵A，若A的转置等于A，则称A为对称矩阵（对角线可以任意数）。
  • 反对称矩阵：对于n阶方阵A，若A的转置等于-A，则称A为对称矩阵（对角线全为0）。
  • 方阵拥有其对应的行列式。矩阵是一个数表，而行列式是一个数（取自不同行不同列元素乘积的和或差）
  • 伴随矩阵：行列式|A|各个元素的代数余子式构成的矩阵的转置（为了求逆矩阵）。

    逆矩阵

  • 对于n阶矩阵A，如果存在n阶矩阵B，使得AB=BA=E，则称A可逆，且B称为A的逆矩阵。同理，称B可逆，A为B的逆矩阵。
  • 如果A可逆，则A的行列式不等于0。
  • 若|A|不等于0，则A可逆，且逆矩阵为A*/|A|。
  • 若|A|≠0，称A为非奇异矩阵（非退化矩阵）。
  • 若A为n阶方阵，若AB=E（或BA=E），则A（B）可逆。
  • 若A为可逆矩阵，则其逆矩阵唯一。
  • 若A为可逆矩阵，则|A^-1^| =1/|A|。
  • 若A可逆，则A^T^也可逆，且(A^T^)^-1^=(A^-1^)^T^.
  • 若A可逆，对k≠0，KA也可逆，且（KA^-1^=1/k×A^-1^）。
  • 若A，B为n阶可逆矩阵，则AB可逆，且（AB）^-1^=B^-1^A^-1^。
  • 在解矩阵方程过程中，可以通过对矩阵进行左乘或右乘相应逆矩阵，达到求解目的。

    矩阵多项式

  • 令A为n阶矩阵，记y = a0E + a1A + a2A^2^ + … + amA^m^，称y为A的m次多项式。
  • 矩阵多项式是可交换的。

    克拉默法则

  • 含有n个未知量x1,x2,x3,…,xn的n个线性方程组称为n阶非齐次线性方程组（它是一个方阵）。
  • 克拉默法则：如果线性方程组的系数矩阵A的行列式不为零，那么方程组有唯一解：x1 = |A1|/|A|, x2 = |A2|/|A|…其中，A1，A2…Aj是系数矩阵第j列元素用方程式右端常数项代替的结果。
  • 若非齐次线性方程组无解或有两个或两个以上解，则系数矩阵为0；
  • 若该方程组为齐次线性方程组，当系数矩阵的行列式不等于0时，该方程只有0解；
  • 当齐次线性方程组有非零解时，则系数矩阵的行列式等于零。

    矩阵分块法

  • 把矩阵A用若干的纵线和横线划分成很多小矩阵，每个小块矩阵称为A的子块。

    矩阵的初等变换与线性方程组

    矩阵的初等变换

  • 把一个矩阵变成一个新的矩阵，下面三种变换称为初等行变换：1.对调矩阵两行；2.以非零数乘以某一行所有元素；3.把某一行所有元素的k倍加到另一行（也可以对列操作）。我们把这六种操作统称为矩阵的初等变换。
  • 初等变换的逆变换：若矩阵A经过一次初等变换变为矩阵B，另一个初等变换又把矩阵B变为矩阵A，称后一个初等变换是前一个初等变换的逆变换。
  • 矩阵的等价：若矩阵A经过有限次初等行（列）变换，变成矩阵B，称A与B行（列）等价。
  • 矩阵等价的性质：反身性，矩阵自己与自己等价；对称性，若A等价于B，则B等价于A；传递性。若A相似B，B相似C，则A和C相似。

    几个特殊的矩阵

  • 行阶梯矩阵：矩阵中可以画一条楼梯一样的阶梯线，每个阶梯只有一行，阶梯右边第一个元素是非零元素。
  • 行最简型矩阵：是行阶梯矩阵，只是阶梯右边第一个元素是一，且一的上面所有元素都是0。
  • 标准型矩阵（矩阵的标准型）：矩阵A经过行列变换转换为含有分块单位矩阵和零元素的矩阵。

    初等矩阵

  • 对于单位矩阵E，经过一次初等变换得到的矩阵，称为初等矩阵（交换行列，第i行/第i列乘以k，第i列的k倍加到第j列/第j行的k倍加到第i行）。
  • 初等矩阵与初等变换的关系：对矩阵A左乘初等矩阵，相当于对A进行一次行初等变换；对矩阵A右乘初等矩阵，相当于对A进行一次列初等变换。
  • 初等矩阵可以辅助求矩阵的逆，如果方阵A可逆，则存在有限个初等矩阵，它们的乘积等于A。
  • 若A可逆，则A只进行行变换或列变换即可得到标准型。
  • 设A与B为m×n阶矩阵，则若A与B行等价，则存在m阶可逆矩阵P，使得PA=B；
  • 设A与B为m×n阶矩阵，则若A与B列等价，则存在n阶可逆矩阵P，使得AP=B；
  • 设A与B为m×n阶矩阵，则若A与B等价，则存在m阶可逆矩阵P，n阶可逆矩阵Q，使得PAQ=B；
  • 假设需要求A的逆矩阵，可以构造分块矩阵（A，E），对其进行行变换，当A变换为标准型矩阵时，右分块则为A^-1^。

    矩阵的秩

  • 在m×n矩阵中，任取k行列位于这些行列交叉处的k^2^个元素，不改变元素的相对位置，而得到的k阶行列式，称为A的k阶子式。
  • 设在矩阵中有一个不等于零的r阶子式D，且所有的r+1阶子式全等于零，那么称D为矩阵A的最高阶非零子式，r称为矩阵A的秩。
  • 对于n阶矩阵A，n阶子式确定：若A的行列式不等于0，则A的秩等于n；若A的行列式等于0，则A的秩小于n。
  • 若A可逆，则A的行列式不等于0，等价于A的秩等于n。
  • 矩阵可分为行满秩矩阵或列满秩矩阵。
  • 对于行阶梯矩阵，矩阵的秩即为非零行的行数。
  • 若A与B等价，则A的秩等于B的秩。
  • max{R(A),R(B)}≤R(A,B)≤R(A)+R(B)
  • R(A±B)≤R(A)+R(B)
  • R(AB)≤min{R(A),R(B)}
  • 若AmnBnt=0，则R(A)+R(B)≤n

    线性方程组的解

  • 若系数矩阵的秩等于增广矩阵的秩等于未知数个数，则方程组有唯一解。
  • 若系数矩阵的秩等于增广矩阵的秩小于未知数个数，则方程组有无穷多解。
  • 若系数矩阵的秩小于增广矩阵的秩，则方程组无解。
  • 若系数矩阵为方阵，克拉默法则也适用。
  • m个方程组的几何意义是代表m个空间平面，求出来的解表示m个平面交于一点。
  • 解方程组一定不能进行列初等变换。
  • 方程的解分为自由变量与非自由变量，两者的个数之和等于未知数的个数。
  • 对于n元齐次线性方程组，若系数矩阵的秩小于n，则方程组有非0解。

    向量组的线性相关性

    向量组及其线性组合

  • n个有次序的数组成的数组称为n维向量，n个数称为向量的n个分量。
  • 向量组：若干个同维数的行向量（列向量）组成的集合叫做向量组。
  • 矩阵可以写成行向量或列向量的形式。
  • 向量b能由向量组A线性表示：对于向量组a1,a2,…，am, k1a1 + k2a2 + …+ kmam称为向量组A的线性组合，k1，k2…km称为组合系数。若b=k1a1 + k2a2 + …+ kmam，则称b可以由a1,a2,…，am线性表示（线性表出）。
  • b可以由向量组A线性表示，即AX=b方程组有解。
  • 向量b能由向量组A线性表示等价于R（A）=R（A，b）。
  • 若向量组B中的每个向量都可以由向量组A线性表示，则称向量组B可以由向量组A线性表示。
  • 若AB=C，则C的列向量组可以由A的列向量组线性表出，C的行向量组可以由B的行向量组线性表出。
  • 若向量组B可由向量组A线性表示，则矩阵方程AX=B有解，即R（A）=R（A，B）。
  • 若向量组A与向量组B可以互相表示，称向量组A与向量组B等价。
  • 向量组A与B等价，则R（A）=R（B）=R（A，B）。
  • 若矩阵A与B列等价，则A与B列向量组也等价，反之不成立。
  • 若矩阵A与B行等价，则A，B行向量组等价，反之不成立。
  • 矩阵A与B等价，不能推出A与B的行和列向量组等价。
  • 若向量组B可以由向量组A线性表示，则R（B）≤R（A）。
  • n阶单位矩阵En=（e1，e2，…，en）的列向量，叫做n维单位坐标向量。
  • 若n维单位坐标向量能由向量组A线性表示，则R（A）=n。
